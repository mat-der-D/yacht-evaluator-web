# ヨット (Yacht) ルール

## 概要

ヨットは5つのサイコロを使って特定の組み合わせを作り、得点を競うダイスゲームです。
「世界のアソビ大全51」に収録されているバージョンのルールを説明します。

## ゲームの流れ

### 基本ルール

- プレイヤーは交互に手番を行う
- 各プレイヤーは12ターンでゲームを終了する
- 全12個の役を1つずつ埋めていく
- 最終的な合計得点が最も高いプレイヤーが勝利

### 1ターンの流れ

1. **サイコロを振る**: 5つのサイコロを振る
2. **サイコロを選択**: 残したいサイコロを選び、残りを振り直す（任意）
3. **再度振り直し**: さらに振り直しができる（任意）
   - 合計で最大3回まで振ることができる
   - 振り直しは2回まで（初回を含めて3回）
4. **役を選択**: 出た目から役を1つ選んで得点を記録
   - 各役は1回しか使えない
   - 得点が0点でも役を埋める必要がある

## 役と得点

### 数字の役（1〜6の目）

それぞれの目の数字が出た個数に応じて得点となる。

| 役名 | 条件 | 得点計算 |
|------|------|----------|
| エース (Ace) | 1の目の個数 | 出た1の目の数 × 1点 |
| デュース (Deuce) | 2の目の個数 | 出た2の目の数 × 2点 |
| トレイ (Trey) | 3の目の個数 | 出た3の目の数 × 3点 |
| フォー (Four) | 4の目の個数 | 出た4の目の数 × 4点 |
| ファイブ (Five) | 5の目の個数 | 出た5の目の数 × 5点 |
| シックス (Six) | 6の目の個数 | 出た6の目の数 × 6点 |

**ボーナス**: エース～シックスの合計得点が63点以上の場合、ボーナス35点を獲得

**例**: サイコロの目が [5, 5, 5, 2, 3] の場合
- ファイブを選ぶと: 5 × 3 = 15点

### 特殊な役

| 役名 | 条件 | 得点 |
|------|------|------|
| **チョイス** | どんな出目でもOK | 5つのサイコロの目の合計 |
| **フォーダイス** | 同じ目が4つ以上 | 5つのサイコロの目の合計 |
| **フルハウス** | 3つ同じ目 + 2つ同じ目 | 5つのサイコロの目の合計 |
| **S.ストレート** | 4つ連続した目 | 15点（固定） |
| **B.ストレート** | 5つ連続した目 | 30点（固定） |
| **ヨット** | 5つすべて同じ目 | 50点（固定） |

### 役の詳細説明

#### チョイス (Choice)
- どんな出目でも選択可能
- 5つのサイコロの合計が得点

**例**: [6, 6, 5, 4, 3] → 24点

#### フォーダイス (Four of a Kind)
- 同じ目が4つ以上ある
- 条件を満たす場合、5つのサイコロの合計が得点
- 条件を満たさない場合は0点

**例**: [4, 4, 4, 4, 2] → 18点

#### フルハウス (Full House)
- 同じ目が3つ + 別の同じ目が2つ
- 条件を満たす場合、5つのサイコロの合計が得点
- 条件を満たさない場合は0点
- **注意**: 5つすべて同じ目の場合も条件を満たし、合計点となる

**例**: [3, 3, 3, 5, 5] → 19点
**例**: [3, 3, 3, 3, 3] → 15点（5つ同じでも有効）

#### S.ストレート (Small Straight)
- 4つ連続した目がある
- [1,2,3,4] または [2,3,4,5] または [3,4,5,6]
- 条件を満たすと15点（固定）

**例**: [1, 2, 3, 4, 6] → 15点

#### B.ストレート (Big Straight)
- 5つすべてが連続した目
- [1,2,3,4,5] または [2,3,4,5,6]
- 条件を満たすと30点（固定）

**例**: [2, 3, 4, 5, 6] → 30点

#### ヨット (Yacht)
- 5つすべてが同じ目
- 条件を満たすと50点（固定）

**例**: [4, 4, 4, 4, 4] → 50点

## 戦略のヒント

### 基本戦略

1. **上段（エース〜シックス）のバランス**
   - 序盤は数字の大きい目（フォー, ファイブ, シックス）を優先的に狙う
   - 小さい目（エース, デュース, トレイ）は後回しでも良い

2. **高得点役を狙う**
   - ヨット（50点）、B.ストレート（30点）は非常に高得点
   - フルハウスやフォーカードも狙いやすく高得点

3. **チョイスの使い方**
   - 失敗した時の保険として温存
   - または確実に高得点を取れる時に使う

4. **振り直しの判断**
   - 1回目で良い出目が出たら、無理に振り直さない
   - 残り振り直し回数を考慮して、リスクを取るか判断

### 上級者向け戦略

1. **期待値計算**
   - 現在の出目と残り振り直し回数から期待値を計算
   - 確実な得点と期待値を比較して判断

2. **役の埋め方**
   - 序盤で高得点役を狙いすぎると、後半で0点埋めが増える
   - バランスよく役を埋めていく

3. **相手の状況を見る**
   - 対戦相手の得点状況を確認
   - 勝つために必要な得点を逆算

## 参考情報

- ゲーム名: ヨット (Yacht)
- プレイ人数: 1〜4人
- プレイ時間: 約10〜20分
- 対象年齢: 全年齢

このゲームは「ヤッツィー (Yahtzee)」と呼ばれるゲームの原型となったゲームです。
